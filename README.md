# MS51FB9AE_KEIL

This project consists of coding for different peripherals of the MS51FB9AE Nuvoton microcontroller.

**Components used:**
- **Microcontroller**
- **Wires**
- **Potentiometer**
- **ESP8266**
- **Logic Analyzer**
- **Programmer**

**Softwares Used:**
- **Keil**
- **Docklight**
- **Logic 2.0.0**
